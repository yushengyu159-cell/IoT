| 字段名 | 数据源 | 说明 |

### 注册时间格式化
```javascript
// 格式化注册时间
let registerTime = user.RegisterTime || '';
if (registerTime) {
    try {
        const date = new Date(registerTime);
        registerTime = date.toLocaleString('zh-CN');
    } catch (e) {
        registerTime = user.RegisterTime;
    }
}
```
