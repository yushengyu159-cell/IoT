# Deploy.sh 优化说明

## 优化版本

**文件**: deploy-with-retry.sh  
**版本**: v2.0  
**状态**: 已优化并测试

---

## 优化功能

### 1. 自动重试机制
- 最大重试次数: 2次
- 重试延迟: 5秒
- 自动恢复临时网络故障

### 2. 国内Go代理
- 代理地址: https://goproxy.cn
- 备用代理: https://goproxy.io
- 作用: 加速Go模块下载

### 3. 详细日志
- 带时间戳的日志输出
- 成功/失败状态显示
- 故障排查建议

---

## 使用方法

### 方式1: 优化脚本（推荐）
```bash
./deploy-with-retry.sh
```

**优点**:
- 自动重试失败的操作
- 使用国内Go代理加速
- 详细的日志输出
- 友好的错误提示

### 方式2: 原始脚本
```bash
./deploy.sh
```

**特点**:
- 保持原有功能不变
- 无重试机制
- 无Go代理优化
