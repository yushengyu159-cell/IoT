# 三种语言转换功能实现记录

## 项目概述
为ESG VISA系统实现完整的多语言支持，包括英文、简体中文、繁体中文三种语言的动态切换功能。

## 实现时间
2025-01-22

## 功能范围

### 1. 前端页面多语言支持
- **index.html** - 登录页面
- **register-1.html** - 注册第一步
- **register-2.html** - 注册第二步  
- **verification.html** - 邮箱验证页面
- **register-3-1.html** - 建筑位置页面
- **location-selector.html** - 位置选择器页面
- **building-type.html** - 建筑类型选择页面
- **role-selection.html** - 角色选择页面
- **control1.html** - 权限管理页面

### 2. 后端邮件多语言支持
- **注册验证邮件** - 三种语言的验证码邮件
- **管理员审核邮件** - 审核通过/拒绝通知邮件
- **密码重置邮件** - 密码重置验证码邮件

## 技术架构

### 前端多语言系统
```
language-config.js (统一语言配置)
├── 语言数据存储 (en, zh-cn, zh-tw)
├── 自动翻译函数 (autoTranslateElements)

└── 语言切换机制 (switchLanguage)

```

```
email.go (邮件服务)

├── 模板选择函数 (GetEmailTemplate)
└── 多语言发送函数 (WithLanguage系列)
```

## 核心文件

### 1. 语言配置文件
**文件**: `/static/language-config.js`
- 包含三种语言的完整翻译数据
- 支持动态语言切换
- 自动应用到所有页面元素

**关键功能**:
```javascript
// 语言切换
LanguageConfig.switchLanguage(lang)

// 获取翻译文本
LanguageConfig.getText(key)

// 自动翻译元素
LanguageConfig.autoTranslateElements(langData)
```

### 2. 邮件服务文件
**文件**: `/internal/service/email.go`
- 多语言邮件模板定义
- 用户语言偏好获取
- 动态邮件内容生成

**关键结构**:
```go
type EmailTemplate struct {
    Subject string
    Body    string
}

type EmailTemplates map[string]EmailTemplate
```

## 实现细节

### 前端实现
1. **页面标识**: 每个页面设置`data-ui-id`属性
2. **翻译属性**: 文本元素添加`data-translate`属性
3. **动态翻译**: JavaScript中使用`tr()`函数
4. **语言存储**: 使用localStorage保存用户语言选择

### 后端实现
1. **数据库字段**: `dids`表添加`Language`字段
2. **API参数**: 邮件发送接口支持`language`参数
3. **模板系统**: 预定义三种语言的邮件模板
4. **语言检测**: 自动获取用户语言偏好

## 翻译键命名规范

### 页面翻译键
```
{页面}.{元素}.{属性}
例如: location.search.placeholder
```

### 邮件翻译键
```
{类型}.{语言}.{模板类型}
例如: verification.en.subject
```

## 使用方式

### 前端语言切换
1. 用户在index页面选择语言
2. 语言设置保存到localStorage
3. 所有页面自动应用选择的语言
4. 无需页面刷新，实时切换

### 后端邮件发送
```go
// 发送多语言验证邮件
SendVerificationEmailWithLanguage(email, language)

// 发送多语言审核邮件
SendAuditResultEmailWithLanguage(email, approved, reason, language)

// 发送多语言密码重置邮件
SendPasswordResetEmailWithLanguage(email, language)
```
